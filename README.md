# Delightful Compatibility
This is a dedicated mod which adds compatibility with various mods in order to make a coherent modded experience

This is built for version 1.20.1 of Minecraft with no expectation for development of other version

This mod is currently NON-FUNCTIONAL and is still in a heavy development phase.
This will stay in version 0.0.0 for a while until a cohesive format is found
